# Hirviurheilu - tulospalvelu Suomen metsästäjäliiton urheilulajeille

## Palvelu on käytettävissä osoitteessa

[https://www.hirviurheilu.com](https://www.hirviurheilu.com)

## Development

### Prerequisites

* [rvm](https://rvm.io/) for Ruby
* [nvm](https://github.com/nvm-sh/nvm) for Node

### Setup

* `nvm use`
* `nvm install`
* `bundle install`

### Start local server

* `bin/dev`

### Tests

* `./run_all_tests.sh`
